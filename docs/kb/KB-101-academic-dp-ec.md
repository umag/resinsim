# Academic Dp/Ec measurements (5 commercial resins)

From "Measuring UV Curing Parameters of Commercial Photopolymers" (PMC5828039).

| Resin | Dp 365nm (µm) | Ec 365nm (mJ/cm²) | Dp 405nm (µm) | Ec 405nm (mJ/cm²) |
|-------|--------------|-------------------|--------------|-------------------|
| PR48 | 42 | 18.3 | 53 | 6.3 |
| VeroWhitePlus | 43 | 0.7 | 145 | 1.9 |
| Formlabs Clear | 146 | 5.2 | 192 | 12.6 |
| TangoBlackPlus | 95 | 2.4 | 151 | 4.1 |
| VeroClear | 186 | 2.1 | 568 | 6.9 |

Key observations:
- Dp range: 42-568µm (13× variation across resins at 405nm)
- Ec range: 0.7-18.3 mJ/cm² at 365nm
- VeroClear has exceptionally high Dp at 405nm (568µm) — nearly transparent
- PR48 has unusually low Dp (42-53µm) — heavily pigmented/absorbing
