# Source: Rapid DLP via a soft, deformable hydrogel separation interface

**"Rapid digital light 3D printing enabled by a soft and deformable hydrogel
separation interface," *Nature Communications* 2021, 12:6070
(s41467-021-26386-6).**

## What it is

A soft-interface separation study — the primary source for the hydrogel
force-reduction figure.

## Key data

- A soft, deformable **hydrogel separation interface reduces separation force to
  ~1/3 of commercially used FEP** at the same printing speed.
- The compliant interface promotes edge-initiated peel (fracture regime) rather
  than whole-area suction ⇒ lower force, faster printing.
